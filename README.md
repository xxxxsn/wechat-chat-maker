# 微信对话生成器

一款**纯前端**的在线微信聊天对话制作工具，可自定义状态栏、多角色对话、文字/图片/语音/红包/转账等，支持截图保存与长图导出。仅供**学习交流与娱乐**使用。

---

## 功能简介

- **外观设置**：手机信号、网络类型、电量、时间、听筒模式、聊天标题、背景图等
- **对话设置**：多用户角色、头像与昵称，发送文字、图片、语音、时间戳、红包、转账等
- **预览与导出**：实时预览手机界面，支持「最新一屏」截图与「全部内容」长图保存

本地打开 `index.html` 或部署到任意静态服务器即可使用，无需后端。

---

## 使用方式

1. 用浏览器直接打开项目根目录下的 `index.html`，或通过任意静态服务器（如 `npx serve .`）访问
2. 在「外观设置」与「对话设置」中编辑内容
3. 在右侧手机预览区查看效果，使用「截图保存（最新一屏）」或「截取全部（长图）」导出图片
